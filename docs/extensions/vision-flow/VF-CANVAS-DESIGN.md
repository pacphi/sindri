# VF-Canvas-Design

Design system framework with fonts and brand guidelines.

## Overview

| Property         | Value                  |
| ---------------- | ---------------------- |
| **Category**     | desktop                |
| **Version**      | 1.0.0                  |
| **Installation** | script                 |
| **Disk Space**   | 200 MB                 |
| **Memory**       | 512 MB                 |
| **Dependencies** | [nodejs](../NODEJS.md) |

## Description

Design system framework with fonts and brand guidelines (from VisionFlow) - provides 36+ font families and comprehensive brand guidelines for design work.

## Network Requirements

- `registry.npmjs.org` - NPM registry

## Installation

```bash
extension-manager install vf-canvas-design
```

## Validation

```bash
test -d ~/extensions/vf-canvas-design
```

## Upgrade

**Strategy:** reinstall

```bash
extension-manager upgrade vf-canvas-design
```

## Removal

```bash
extension-manager remove vf-canvas-design
```

Removes:

- `~/extensions/vf-canvas-design`
