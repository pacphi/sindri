# VF-Algorithmic-Art

Generative algorithmic art with p5.js patterns.

## Overview

| Property         | Value                  |
| ---------------- | ---------------------- |
| **Category**     | utilities              |
| **Version**      | 1.0.0                  |
| **Installation** | script                 |
| **Disk Space**   | 100 MB                 |
| **Memory**       | 256 MB                 |
| **Dependencies** | [python](../PYTHON.md) |

## Description

Generative algorithmic art with p5.js patterns (from VisionFlow) - creates generative art using algorithmic patterns.

| Tool                                    | Type    | Description    |

| Various Python art generation libraries | library | Art generation |

## Installation

```bash
extension-manager install vf-algorithmic-art
```

## Validation

```bash
test -d ~/extensions/vf-algorithmic-art
```

## Upgrade

**Strategy:** reinstall

```bash
extension-manager upgrade vf-algorithmic-art
```

## Removal

```bash
extension-manager remove vf-algorithmic-art
```

Removes:

- `~/extensions/vf-algorithmic-art`
